                         『ミニッツマスコット』 について                                                                    ver 1.4                                                          2001/5/21 by ueno○ミニッツマスコットとは　　画面に、マスコットを表示します.......それだけです。　　データをユーザーが作成することも可能です。　　○使い方　　解凍したファイルを、フォルダごとそのままCEに転送します。　　マスコットデータが複数ある場合は選択ダイアログが開きますので、ダブルタップにて　　選択してください。○マスコットデータについて　　マスコットデータは、フォルダごとにMMascot.exeと同じフォルダに入れてください。　　その際、skin.txtがMMascot.exeの一階層下のフォルダにあるようにしてください。　　最大のマスコットデータ数は、32個です。　　マスコットデータは、ユーザーが作成することも可能です。作成方法は、データ作成マニュアルを　　参照してください。○操作方法　　タスクトレイのアイコンをタップすると、画像の表示・非表示を切り替えられます。　　タスクトレイのアイコンのダプルタップまたは二重起動でダイアログが開きます。　　　・U,D,L,R : それぞれ上、下、左、右に表示位置を1ピクセル移動します　　　・Ux10,Dx10,Lx10,Rx10 : それぞれ上、下、左、右に表示位置を10ピクセル移動します　　　　・Hide : 画像を非表示にします　　　・透過度 : 画像表示の透過度を100%,75%,50%,25%,Halfから選択できます                 Halfは間引き表示です。256色機では100%とHalfのみとなります　　　・About : バージョン情報を表示します　　　・Quit : 終了します○注意　　実行するとMMascot.exeと同じフォルダに __tmp というファイルが出来ますが、　　実行中は削除しないでください。○課題　　●将来的に搭載予定の事項　　　・マスコットデータの切り替え（必要であれば）　　●改善が不可能な事項　　　・PIEやメモ帳でスクロールさせると、ごみが表示されることがある　　　・アプリの切り替え等で画面を再描画させると、ごみが表示されることがある　　　　・画像のタップやドラッグを使用できない　　　・ロードした画像と異なる色で表示されることがある○注意事項　　・動作確認はしていますが、このプログラムを実行したことによる不具合に関しては　　　作者は一切の責任を負いません。事前にバックアップ等を取る等しておいてください。　　・データの作成・配布に制約は設けません。作成されたデータの権利は、データ作成者の　　　所有するものとなります。　　・当プログラムはフリーウエアとします。デモ・配布などに制約はありません。　　　ただし、著作権は放棄していません。また、当プログラムおよびドキュメントの　　　改変および商用目的の使用は禁止いたします。　　・当プログラムの雑誌への掲載、作成したデータへの添付等については、ご連絡ください。　　　個別に検討させていただきます。○連絡先　　ご意見、ご要望は、下記までお願いします。　　・コンセプト、サンプルデータについて
